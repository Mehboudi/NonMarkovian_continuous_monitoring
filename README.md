# NonMarkovian Continuous Monitoring

Code accompanying the manuscript:
**"Parameter Estimation in a Continuously Monitored Non-Markovian Quantum System"**

## Repository layout

This repository is organized into two independent code folders:

- `folder_1/` — first code package (to be uploaded by the repository owner)
- `folder_2/` — second code package (to be uploaded by collaborator)

> Replace `folder_1` and `folder_2` with the actual folder names if they differ.

## Collaboration workflow

1. Upload the first code folder into this repository.
2. Your colleague uploads the second code folder.
3. Keep each folder self-contained (its own scripts, modules, and dependencies).
4. Update this README with folder-specific run instructions once both folders are added.

## Notes

- The two folders are intentionally separated to keep contributions independent.
- Shared documentation can remain in this root `README.md`.
